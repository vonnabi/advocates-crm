# Текущие изменения ветки

Дата фиксации: 22.06.2026
Коммит для Render/GitHub: `9da5f21`

## Справи

- В карточке справи удалены лишние поля этапа, дедлайна и приоритета из правого блока.
- Блок органа заменен на `Сторони по справі`.
- Для сторон поддерживается несколько записей в одной справе.
- Стороны показываются компактными карточками: по умолчанию закрыто, детали раскрываются по кнопке.
- Заголовки карточек справи приведены к одному стилю и размеру.
- В списке справ добавлен компактный индикатор количества справ клиента.
- Клик по строке клиента раскрывает/сворачивает его справи.
- В заголовке колонки `Клієнт` добавлен общий переключатель: выключен — сгруппировано, включен — все справи раскрыты.
- Таблица справ перераспределена по ширинам, чтобы клиент, сторона, финансы и действия не обрезались на desktop/mobile.

## Процесуальні дії

- Раздел переосмыслен как журнал событий по справе.
- Колонки изменены на: подія, ініціатор події, дата і час, опис.
- Статус из таблицы процессуальных действий убран.

## Документообіг

- Раздел `Документи` переименован в `Документообіг`.
- Глобальные документы отделены от документов конкретной справи.
- Добавлена логика архивных папок и папки для зразків.
- ONLYOFFICE, подпись и отправка оставлены для документов по справе и процессуальных документов.

## Розсилка

- Экран рассылки упрощен под практичный сценарий: текст, картинка, получатели, канал отправки.
- Интерфейс рассылки приведен к визуальному стилю CRM, карточками и плотными блоками.

## ONLYOFFICE + Render

- Backend формирует ONLYOFFICE config server-side.
- Для Render CRM берет публичный URL из `CRM_PUBLIC_URL`, `RENDER_EXTERNAL_URL` или `RENDER_EXTERNAL_HOSTNAME`.
- `serverAccessUrl`, callback и URL файла для ONLYOFFICE больше не должны уходить как `127.0.0.1` на проде.
- Секрет Docs Cloud берется из `ONLYOFFICE_JWT_SECRET`.
- В `render.yaml` добавлены `CRM_PUBLIC_URL`, `ONLYOFFICE_DOCUMENT_SERVER_URL`, `ONLYOFFICE_JWT_SECRET` как `sync: false`.

## Налаштування

- Аудит готовности теперь нужно держать компактным: в шапке показывается текущий процент, детали раскрываются по клику.
