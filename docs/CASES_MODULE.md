# Модуль: Справи / Дела

## Назначение

Раздел `Справи` является центральным модулем юридической CRM. Через дело связываются клиент, документы, задачи, календарь, финансы, история действий и будущий AI-помощник.

## Поля дела для будущей Django-модели

Основные поля:

- `id`
- `case_number`
- `client`
- `title`
- `case_type`
- `status`
- `stage`
- `priority`
- `responsible_user`
- `court_or_authority`
- `description`
- `opened_at`
- `deadline_at`
- `income_amount`
- `debt_amount`
- `created_at`
- `updated_at`

Связи:

- дело принадлежит клиенту;
- дело имеет много документов;
- дело имеет много задач;
- дело имеет много событий календаря;
- дело имеет много финансовых операций;
- дело имеет историю изменений;
- дело может иметь отдельного AI-помощника.

## Статусы дела

- `В роботі`
- `Очікує відповідь`
- `Терміново`
- `Закрито`
- `Призупинено`

## Этапы дела

Примеры:

- первинна консультація;
- збір доказів;
- підготовка документів;
- подача документів;
- судове засідання;
- очікування відповіді;
- виконання рішення;
- завершено.

## Что уже реализовано в прототипе

- KPI по делам;
- таблица дел;
- поиск по делу, клиенту, ответственному;
- фильтр по типу;
- фильтр по статусу;
- карточка дела справа;
- связь дело -> клиент;
- ответственный сотрудник;
- суд / орган;
- дедлайн;
- приоритет;
- финансовый блок;
- документы по делу;
- задачи по делу;
- календарные события по делу;
- история дела.
- создание нового дела;
- добавление документа в дело.
- редактирование дела через карточку и карандаш в таблице.
- добавление документа как файла с компьютера или ссылки Google Doc/Drive.
- счетчики документов, задач и событий в таблице дел.
- автоматическое раскрытие нужного блока после добавления документа, задачи или события.
- добавление задачи и события прямо из карточки дела.
