# Design System

## Visual Style

Premium modern legal SaaS: calm, dense, structured, readable, and business-focused.

Primary visual reference: [[Dashboard-Style-Reference]].

## Foundations

- Colors: centralized tokens.
- Typography: see [[Typography-System]].
- Spacing: 4, 8, 12, 16, 20, 24, 32, 40, 48.
- Border radius: consistent card, control, modal, and badge radii.
- Icons: one icon style, consistent stroke and size.

## Components

Cards, tables, forms, buttons, badges, sidebar, header, modals, dropdowns, and responsive layouts should use shared tokens.

## Rule

Future visual work should change tokens and reusable classes first, not random one-off component values.

When a module looks inconsistent, compare it with [[Dashboard-Style-Reference]] before adding new styles.
