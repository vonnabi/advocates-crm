# Responsive Rules

## Breakpoints

- 320px
- 375px
- 390px
- 430px
- 768px
- 1024px
- 1280px
- 1440px
- 1920px

## Rules

- Desktop sidebar is visible from 1024px.
- Hamburger drawer is used below 1024px.
- No horizontal scroll.
- Mobile uses one column.
- Tablet uses two columns where appropriate.
- Touch target minimum is 44px.
- Preview/right panels move below content on tablet and mobile.
- Modals fit mobile screens and scroll internally if needed.
