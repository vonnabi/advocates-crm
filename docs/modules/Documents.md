# Documents

## Purpose

Create, store, classify, edit, send, export, and archive documents.

## Main Features

- Document list
- Case document tree
- Procedural documents
- ONLYOFFICE opening/editing
- Archive folders

## Main UI Screens

- Documents page
- Add/edit document modal
- Document preview/right panel

## Data Involved

Document name, type, status, folder, case, client, file URL/path, source, dates.

## Permissions

File access must follow case and role permissions.

## AI Opportunities

Draft generation, classification, summary, export templates.
