# Typography System

## Goal

All text should be controlled from centralized typography tokens.

## Principles

- Use fluid typography with `clamp()`.
- Keep semantic roles: page title, section title, card title, body, body small, caption, label, button, table header, table cell, badge, input.
- Avoid random hardcoded font sizes.
- Make desktop dense and mobile readable.

## Areas To Check

Headings, labels, buttons, tables, modals, menus, badges, inputs, placeholders, helper text, sidebar items, tabs, dropdowns, tooltips, notifications, AI chat, empty states, loading states, and error states.
