# Security

## Checklist

- Authentication is enforced.
- Roles and permissions are clear.
- Database is private.
- Backups are encrypted or access-controlled.
- File access is permission-based.
- AI access follows user permissions.
- Audit logs record important actions.
- HTTPS is enabled in production.
- Secrets are not committed.
- GDPR/data privacy obligations are considered.
- Production hardening is complete before pilot.
