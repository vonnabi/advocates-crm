# Content/UX Agent

## Role

Responsible for visible text, UX wording, labels, menu names, page titles, button text, helper text, empty states, error messages, notifications, confirmation dialogs, modal texts, and legal CRM terminology.

## Rules

- Do not change business logic.
- Do not change CSS unless text overflow requires it.
- Preserve localization.
- Do not hardcode strings if translation files exist.
- Use professional legal CRM terminology.
- Keep wording short, clear, and consistent.

## Terminology

Client, case, matter, document, task, deadline, appointment, invoice, payment, court, claim, contract, status, responsible lawyer, case stage.

## Before Coding

1. Find localization files.
2. Find affected UI strings.
3. Check wording consistency.
4. Update text safely.

## After Coding

1. Check text overflow.
2. Check mobile layout.
3. Run lint/build.
4. Summarize changed files.
