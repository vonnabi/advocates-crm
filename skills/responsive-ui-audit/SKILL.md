# Responsive UI Audit

Use this local skill when checking or improving responsive UI.

## Breakpoints

320px, 375px, 390px, 430px, 768px, 1024px, 1280px, 1440px, 1920px.

## Rules

- No horizontal scroll.
- Sidebar visible on 1024px+.
- Hamburger drawer below 1024px.
- Cards stack correctly.
- Tables become mobile-friendly.
- Modals fit mobile screen.
- Buttons are touch-friendly.
- Minimum touch target: 44px.
- Text does not overflow.
- Forms are usable on mobile.
- Preview panels stack correctly.

## Sidebar Behavior

Desktop 1024px+: full sidebar visible.

Below 1024px: sidebar hidden by default, hamburger visible in header, drawer slides from left, overlay closes drawer, close button closes drawer, menu item click closes drawer on mobile.
