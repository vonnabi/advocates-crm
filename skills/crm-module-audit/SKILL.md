# CRM Module Audit

Use this local skill when auditing or improving one CRM module.

## Steps

1. Identify module.
2. Find all related pages/components.
3. Check data flow.
4. Check UI consistency.
5. Check text consistency.
6. Check responsive behavior.
7. Check loading, empty, and error states.
8. Check modals/dropdowns related to the module.
9. Fix safe issues.
10. Run lint/build.
11. Summarize changed files.

## Modules

Dashboard, Clients, Cases, Documents, Calendar, Tasks, Mailing, Finance, AI Assistants, Settings.
