# CRM Frontend Polish

Use this local skill when visually polishing the CRM frontend.

## Goal

Improve premium SaaS visual quality without changing business logic.

## Check And Improve

Spacing, visual hierarchy, card composition, shadows, borders, border radius, icons, action buttons, menus, modals, dropdowns, forms, tables, empty states, loading states, error states, and responsive layout.

## Design References

Stripe Dashboard, Linear, Attio CRM, Vercel Dashboard, HubSpot.

## Rules

- Do not overdecorate.
- Keep legal-business style.
- Avoid visual noise.
- Make UI cleaner, calmer, and more premium.
