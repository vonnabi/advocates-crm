# Design

Папка для дизайна и референсов.

Сюда стоит перенести скрины из WhatsApp, чтобы они лежали рядом с проектом.

Уже перенесены первые референсы:

- `screenshots/clients.jpg` — база клиентов.
- `screenshots/calendar.jpg` — календарь.

## Что фиксируем по стилю

- темно-синее боковое меню;
- светлая рабочая область;
- таблицы с фильтрами;
- карточка выбранной сущности справа;
- KPI-карточки;
- синие основные кнопки;
- зеленые успешные статусы;
- красные ошибки и риски;
- желтые предупреждения.
