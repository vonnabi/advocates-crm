# Integrations

Папка для описания внешних сервисов.

## Telegram

Понадобится:

- создать Telegram-бота через BotFather;
- сохранить токен в защищенных настройках backend;
- дать клиенту ссылку для подключения;
- хранить `telegram_chat_id`;
- отправлять напоминания и рассылки только тем, кто активировал бота.

## SMS

Понадобится:

- выбрать SMS-провайдера;
- получить API-ключ;
- хранить статусы доставки;
- учитывать стоимость и количество SMS-сегментов.

## Email

Понадобится:

- SMTP или сервис рассылок;
- шаблоны писем;
- статусы доставки;
- ссылка отписки для массовых сообщений.

## AI

Понадобится:

- API-ключ AI-провайдера;
- ограничение доступа по делам;
- хранилище документов;
- журнал запросов;
- генерация документов.
